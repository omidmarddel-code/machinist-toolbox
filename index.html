دستگاه وایرکات ایزی پایپ

تنظیمات وایرکات

نوع قطعه یا عملیات را انتخاب کن تا تنظیمات پیشنهادی نمایش داده شود.

جزوه آموزشی

آموزش اپراتوری دستگاه وایرکات

01

حرکت محورها و بستن قطعه

ابتدا برای بستن قطعه کار مورد نظر، نیاز به حرکت محورها داریم.

MACHINE → MANUAL → X.Y.Z.U.V

سپس قطعه کار را بسته و رفرنس را طبق رفرنس تعیین‌شده توسط برنامه‌نویس می‌گیریم.

02

رفرنس کردن دایره

این دستور برای رفرنس کردن دایره، مانند ماتریس تریم، استفاده می‌شود.

MODE → MAIN → CENTER → 0 → ENTER

دستگاه به صورت اتوماتیک وسط دایره را پیدا کرده و رفرنس را وسط دایره قرار می‌دهد.

03

رفرنس گرفتن از گوشه یا سطح قطعه

برای گرفتن رفرنس در گوشه کار، ابتدا محور مورد نظر را نزدیک کار می‌کنیم، سپس به سمتی که نیاز به رفرنس است، محور X یا Y را با علامت مثبت یا منفی تنظیم می‌کنیم.

MODE → EDG,+-X/Y,X/Y0

این دستور برای رفرنس کردن یک سطح قطعه کار مورد استفاده قرار می‌گیرد.

04

رفرنس وسط محور X و Y

برای رفرنس گرفتن وسط محور X و Y به صورت جداگانه از دستور زیر استفاده می‌کنیم.

MODE → MIDEL → 90 = Y 0 = X → ENTER

سپس دستگاه به صورت جداگانه وسط دو محور را پیدا کرده و رفرنس را وسط قرار می‌دهد.

MODE → SETUP → REFRENCE → X0 Y0 → ENTER

بعد از پیدا کردن موقعیت رفرنس، باید مقدار عدد X و Y را صفر کنیم.

05

حرکت خطی و برش قالب گرد

برای حرکت خطی، ماشین‌کاری و نصف کردن قالب‌های گرد از دستور زیر استفاده می‌کنیم.

برای قالب گرد حدود 140 میلی‌متر، دستور برش به این شکل نوشته می‌شود:

MODE → CTR,X/Y+/-150 → ENTER → START
06

اجرای برنامه از داخل فلش

فلش را در حالت 00 فلاپی قرار داده و سپس مسیر زیر را اجرا می‌کنیم.

FILE → DIRECTORY → DD0: → ENTER

سپس فایل مورد نظر را انتخاب می‌کنیم و برای اجرای آن:

MODE → MACHINE → DISPLAY → CLEAR → START
MACHINE → CUT → START

قبل از استارت برنامه باید درب تانک بسته و آب مخزن پر شود.

07

تنظیمات حین برش

پس از شروع حرکت دستگاه، تنظیمات فشار آب، کشش سیم، ولتاژ جرقه و سایر تنظیمات را طبق جدول مناسب هر قطعه تنظیم می‌کنیم.

MACHINE → GENERATOR → FITTING → INJECTION → +-

تنظیم سرعت پاشش آب

MACHINE → GENERATOR → FITTING → WIRE → WIRE SPEED → +-

تنظیم سرعت سیم

MACHINE → GENERATOR → OTHER → FF OR F → +-

تنظیم پیشروی و قدرت جرقه دستگاه

MACHINE → CUT → COM/SIM,1 = ON COM/SIM,0 = OFF → START

برای رفتن به حالت سیمولیشن

08

نکته‌های نگهداری و ایمنی

  • هر نیم ساعت به دستگاه سر زده و از عملکرد صحیح دستگاه اطمینان حاصل فرمایید.
  • هر روز قبل از شروع به کار، تانک دستگاه را با سنباده نرم تمیز کرده و غبار را جمع کنید.
  • فیلتر دستگاه باید به طور مرتب هر 3 ماه یک‌بار به صورت کامل تعویض گردد.
  • TDS دستگاه نباید از 50 عبور کند. در صورت عبور، رزین دستگاه باید چک شود.

جزوه آموزشی

آموزش اپراتوری دستگاه فرز فانوک سری O-M

01

روشن کردن دستگاه و رفرنس محورها

ابتدا دستگاه را روشن می‌کنیم. قبل از شروع کار، همیشه بعد از روشن کردن دستگاه باید محورها به حالت صفر عدد ماشین برسند تا چراغ مربوط به صفر محورهای X، Y و Z روشن شود.

ZRN → RAPID → AXIS Z → +

ابتدا کلید ZRN را روشن کرده و ولوم را روی RAPID قرار می‌دهیم. سپس ولوم AXIS را روی محور Z گذاشته و کلید مثبت را نگه می‌داریم تا دستگاه به نقطه صفر ماشین حرکت کند.

بعد از روشن شدن چراغ محور Z، همین کار را برای بقیه محورها انجام می‌دهیم تا سه چراغ رفرنس ماشین روشن شود.

نکته: این کار فقط یک بار انجام می‌شود؛ زمانی که دستگاه خاموش بوده و تازه راه‌اندازی شده است.

02

بستن قطعه و شروع رفرنس‌گیری

قطعه کار را داخل گیره بسته و طبق رفرنس اعلام‌شده توسط برنامه‌نویس پیش می‌رویم. برای مثال در قالب گرد 125 تن، رفرنس Y در قسمت فک ثابت و رفرنس X وسط قالب است.

اپراتور باید با استفاده از ابزار تاچ پراپ شروع به رفرنس‌گیری کند. ابتدا تاچ پراپ را با ولوم MDI فراخوانی کرده و ابزار را در محور Y نزدیک قطعه کار می‌کنیم.

ابتدا با مقدار 0.1 به قطعه نزدیک می‌شویم. وقتی بوق اول خورد، 0.1 برمی‌گردیم، هندویل را روی 0.01 تنظیم می‌کنیم و این بار به صورت صدمی نزدیک قطعه می‌شویم.

03

صفر کردن G54 و کنترل ABS

وقتی بوق ممتد شد، باید نصف قطر گوی تاچ پراپ را با استفاده از فضای REL جلو بیاییم.

سپس باید ABS را در G54 یا همان 01 صفر کنیم. برای این منظور عدد ماشین دستگاه را که در قسمت POS ALL قرار دارد، بدون هیچ تغییری در قسمت OFFSET جلوی 01 وارد می‌کنیم.

POS ALL → OFFSET → 01 → RESET

بعد از آن در قسمت POS ALL وقتی کلید RESET را می‌زنیم، عدد ABS باید صفر شود. اگر عدد صفر شد یعنی مسیر درست بوده و عدد ماشین درست وارد شده است.

بقیه محورها نیز به همین صورت رفرنس‌گیری می‌شوند و در آخر باید همه محورها چک شوند.

04

رفرنس محور Z

برای رفرنس رفتن محور Z باید ابزار مرجع مشخص‌شده دستگاه فانوک را فراخوانی کنیم. در این دستگاه ابزار مرجع، ابزار 16 است.

Tool 16 → Z ZERO / Paper 0.03 → OFFSET 01

سپس با استفاده از کاغذ 0.03 یا Z ZERO تاچ کرده و عدد آن را مانند محورهای دیگر در سمت 01 قرار می‌دهیم.

05

افست‌گیری ابزارها

برای افست‌گیری ابزارها به ابزار Z ZERO نیاز داریم. ابتدا ابزار مرجع، یعنی ابزار 16، را فراخوانی کرده و آن را روی Z ZERO در قسمت صفر تنظیم می‌کنیم.

سپس عدد REL Z را صفر می‌کنیم. در این حالت هم در Z ZERO عدد صفر است و هم در محور Z REL.

حالا ابزار مورد نظر را فراخوانی کرده و آن را روی Z ZERO در قسمت صفر تنظیم می‌کنیم. سپس با کلید OFFSET به قسمت افست ابزارها می‌رویم.

OFFSET → Tool Number → Z REL → INPUT

برای مثال اگر ابزار 8 فراخوانی شده باشد، باید عدد Z REL را در قسمت شماره ابزار 8 وارد کرده و کلید INPUT را بزنیم. با این روش افست ابزار 8 نسبت به ابزار مرجع گرفته می‌شود.

06

اجرای برنامه از فلش

بعد از اینکه رفرنس طبق برنامه گرفته شد و افست ابزارها وارد شد، نوبت اجرای برنامه است.

فلش را وارد مبدل دستگاه کرده و با استفاده از قلم مبدل، برنامه مورد نظر را آورده و گزینه SAVE را می‌زنیم.

TAPE → CYS

برای اجرای برنامه از کنترل دستگاه، ولوم را روی حالت TAPE قرار داده و با استفاده از کلید CYS برنامه را اجرا می‌کنیم.

07

نکته‌های اجرای برنامه و ابزار

  • قبل از اجرای برنامه، همه FEED RATE ها بسته باشند و بعد آرام باز شوند تا ابزار در شروع کار به‌آرامی روی قطعه بیاید. سپس مقدار را روی 100 درصد قرار دهید.
  • به دلیل بالا بودن دور ابزار و فیدریت، همیشه باید از سلامت مایع خنک‌کننده مطمئن باشید تا ابزار موقع کار نسوزد.
  • ابزارها همیشه باید تیز باشند و فاز برشی سالمی داشته باشند تا بعد از ماشین‌کاری دچار تغییر سایز نشویم.

جزوه آموزشی

آموزش اپراتوری دستگاه فرز HEIDENHAIN

01

روشن کردن دستگاه و خروج از Emergency

دستگاه را روشن می‌کنیم و کلید CE را می‌زنیم. سپس کلید سفید I را می‌زنیم تا درایو محورها از حالت EMERGENCY خارج شود.

بعد از آن، مانند دستگاه CNC فانوک، دستگاه باید به محل صفر ماشین برود.

02

رفرنس کردن محورها

برای رفرنس کردن، ولوم FEED RATE را روی 100 درصد قرار می‌دهیم و در فضای REFRENCE دستگاه، کلید سبز اصلی CYS را سه بار می‌زنیم.

FEED RATE 100% → REFRENCE → CYS × 3

در این حالت دستگاه ابتدا محور Z، سپس X و در آخر Y را به محل صفر ماشین می‌برد. بعد از اتمام، دستگاه به صورت اتوماتیک وارد فضای POS می‌شود.

نکته: در دستگاه هایدن هاین برخلاف فانوک که فضای REL و ABS دارد، فقط یک فضای POS داریم.

03

تعویض ابزار با MDI

برای تعویض ابزار، مانند دستگاه فانوک وارد حالت MDI می‌شویم، اما دستور را مطابق زبان دستگاه هایدن هاین می‌نویسیم. مثال زیر برای تعویض به ابزار 16 است.

BEGIN PGM $MDI MM
TOOL CALL 16 Z S200:
STOP M6
END PGM $MDI MM

با این دستور، دستگاه تعویض ابزار به ابزار 16 را شروع می‌کند.

نکته: برای ادیت برنامه، کلید راست ادیت را فعال و کلید چپ ادیت را غیر فعال می‌کنیم.

04

رفرنس گرفتن قطعه

ابتدا با دستور گفته‌شده از MDI، ابزار تاچ پراپ را فراخوانی می‌کنیم. سپس با استفاده از کلیدهای RAPID و محورهای X، Y و Z، ابزار را به قطعه نزدیک می‌کنیم.

MDI → Touch Probe → RAPID X/Y/Z

بعد مطابق مطالب گفته‌شده در جزوه فانوک، تاچ می‌کنیم. برای صفر کردن محورهای X، Y و Z باید از کلید DATUM SET استفاده کنیم.

DATUM SET → Axis X/Y/Z → 0 → ENT
05

افست‌گیری ابزارها

ابتدا مانند دستگاه فانوک، ابزار مرجع را می‌آوریم. در فرز هایدن هاین نیز ابزار مرجع، ابزار 16 است.

با Z ZERO عدد Z را با گزینه DATUM SET صفر می‌کنیم. سپس ابزار بعدی را آورده و با Z ZERO تاچ می‌گیریم.

Tool 16 → Z ZERO → DATUM SET

بعد با استفاده از گزینه TOOL TABLE فضای افست باز می‌شود. با فعال کردن گزینه EDIT روی شماره ابزار مورد نظر می‌رویم.

TOOL TABLE → EDIT → Tool Number → L → _!_ → END

در قسمت L، که همان طول ابزار است، با زدن یک بار کلید _!_ عدد L تغییر می‌کند. سپس با گزینه END از فضای افست خارج می‌شویم. بقیه ابزارها نیز به همین روش افست‌گیری می‌شوند.

06

انتقال و اجرای برنامه

با استفاده از کلید مربوطه و گزینه WINDOW، ابتدا در قسمت چپ پنجره وارد RS232 می‌شویم.

WINDOW → RS232 → COPY → TNC

برنامه مورد نظر را با گزینه COPY وارد هارد دستگاه، که TNC نام دارد، می‌کنیم.

بعد از آن وارد فضای اجرای برنامه شده، روی برنامه مورد نظر ENT می‌زنیم و برنامه را با گزینه CYS اجرا می‌کنیم.

Program → ENT → CYS

جزوه آموزشی

آموزش اپراتوری دستگاه تراش CNC GSK

01

روشن کردن دستگاه و رفرنس محورها

ابتدا دستگاه را روشن می‌کنیم. قبل از شروع کار همیشه بعد از روشن کردن دستگاه باید محورهای X و Z به صفر ماشین برسند تا چراغ مربوط به صفر محورها روشن شود.

MACHINE ZERO → X → Z

دستگاه به صورت اتوماتیک به صفر ماشین رسیده و آماده کار خواهد بود.

02

آشنایی با REL ، ABS و فضای MDI

در قسمت POSITION دو پنجره مهم REL و ABS وجود دارد.

REL برای صفر کردن دستی محورها و ABS برای تنظیم افست ابزارها و اجرای برنامه استفاده می‌شود.

در فضای MDI می‌توان دستورات تک خطی اجرا کرد.

G97 S500 M03 → START

روشن کردن اسپیندل ساعتگرد با دور ثابت 500 دور.

03

OFFSET گرفتن ابزار مرجع T808

ابزار مرجع دستگاه، ابزار T808 است.

ابزار را توسط MDI فراخوانی کرده، به قطعه نزدیک کنید و پیشانی تراشی انجام دهید.

سپس محور Z را مطابق رفرنس برنامه صفر کنید. اگر رفرنس وسط نوک ابزار باشد، ابتدا مقدار شعاع نوک ابزار را در REL جبران کرده و سپس ABS را صفر نمایید.

مقدار صفر را در OFFSET قسمت Z ابزار 00 وارد کرده و سپس مقدار معکوس عدد ABS را ثبت کنید تا محور Z دقیقاً صفر شود.

محور X نیز به همین روش تنظیم می‌شود.

04

OFFSET گرفتن سایر ابزارها

ابزارهای دیگر را به صورت دستی فراخوانی کنید.

با تاچ کردن ابزار به قطعه، مقادیر X و Z موجود در ABS را بدون تغییر در قسمت OFFSET همان ابزار وارد نمایید.

تمام ابزارها نسبت به ابزار مرجع T808 افست گرفته می‌شوند.

05

اجرای برنامه از فلش

فلش را به دستگاه متصل کنید.

PROGRAM → USB

برنامه مورد نظر را انتخاب کنید.

AUTO → CYCLE START

دستگاه اجرای برنامه را آغاز می‌کند.

06

نکات مهم اپراتوری

  • قبل از شروع برنامه از تیز بودن ابزارها اطمینان حاصل کنید.
  • آب صابون را به درستی روی ابزار تنظیم کنید.
  • سرعت اسپیندل و پیشروی را متناسب با قطر قطعه تنظیم نمایید.
  • اجرای چند قطعه اول را کاملاً کنترل کنید.
  • به صورت مرتب دستگاه را بررسی کنید تا سفاله دور اسپیندل نپیچد.
  • در صورت پیچیدن سفاله، دستگاه را متوقف کرده و سپس ادامه برنامه را اجرا نمایید.
